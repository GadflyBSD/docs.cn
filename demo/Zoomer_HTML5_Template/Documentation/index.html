Additional css files at css folder

  1. bootstrap.css - Twitter Boostrap CSS framework
  2. icoMoon.css - Icons Font
  3. prettyPhoto.css - style for prettyphoto jQuery plugin
  4. flexslider.css - style for FlexSlider jQuery plugin
  5. drop-down.css - style for Drop-down list
  6. colorbox.css - style for lightbox
  7. camera.css - style for Camera Slider jQuery plugin
  8. bootstrap-responsive.css - Twitter Boostrap CSS framework responsive
  9. responsive.css - Responsive CSS

JavaScript

This theme imports following Javascript files.

  1. jquery-1.7.2.min.js - Javascript library
  2. retina.js - Retina display jQuery plugin
  3. custom.js - contains all custom script
  4. bootstrap.js - Twitter boostrap framework
  5. bootstrap-tooltip.js - Boostrap tooltip
  6. jquery.touchSwipe.min.js - jQuery plugin for touch devices
  7. jquery.sticky.js - jQuery plugin for sticky
  8. jquery.parallax-1.1.3.js - jQuery plugin for parallax effect
  9. jquery.localscroll-1.2.7-min.js - jQuery plugin for scroll effect
  10. jquery.scrollTo-1.4.2-min.js - jQuery plugin for scroll effect
  11. jquery.isotope.min.js - An exquisite jQuery plugin for magical layouts
  12. jquery.carouFredSel-6.2.1-packed.js - FredCarousel slider
  13. toucheffects.js - jQuery plugin for touch devices
  14. jquery.prettyPhoto.js - prettyphoto jQuery plugin
  15. jquery.form.js - allows you to upgrade HTML forms to use AJAX
  16. jquery.validate.min.js - validate contact form and comment form
  17. tweetable.jquery.js - jQuery plugin for Twitter
  18. jquery.timeago.js - jQuery plugin for Twitter
  19. jquery.colorbox.js - jQuery plugin for lightbox
  20. jquery.flexslider-min.js - jQuery plugin for Flex slider
  21. classie.js - class helper functions from bonzo
  22. cbpAnimatedHeader.js - jQuery plugin for Header effect when scroll
  23. cbpScroller.js - jQuery plugin for scroll effect
  24. jquery.mobile.customized.min.js - jQuery Mobile framework customized for Camera slideshow
  25. jquery.easing.1.3.js - jQuery Easing plugin
  26. camera.min.js - jQuery plugin for Camera Slider
  27. modernizr.custom.js - jQuery Modernizr
  28. main.js - Contain some custom script
  29. http://maps.google.com/maps/api/js?sensor=false - Google map jQuery plugin

Except custom.js, all of the plugins are packed, so you don't need to manually edit anything in these files. The only necessary thing to know is how to call the method. You can refer to the website of the plugin author I listed in the "Sources and Credits" section to learn more about them.
